Setting up rotations

Optionally setup and remove rotations to be used throughout the regatta.

Introduction

Rotations can be used to enter finishes by using the sail numbers instead of the actual team names. They can be very complex in order to promote fairness. Each team is assigned a boat from a fixed set for each race in each division. If the scoring is combined, each team in each division will receive a unique boat in each race. This pane provides a number of parameters you can tweak to create virtually any rotation desired.

Parameters

Typically, rotations are created for sets of races at a time. With the parameters below, it is possible to create an entire rotation with just a few clicks, as TechScore takes care of shuffling the sail numbers for each race.

Rotation Type

As a team's assigned sail shifts from one race to another, there are three ways to determine the next sail:

  • Standard (+1) assigns the next available sail from the set. If sails are numbered sequentially, this results in a team always moving to the boat with its current sail number plus 1. When the last sail is reached, the next sail becomes the first in the list.
  • Direct swap either moves the team into the next sail from the set or the previous one in such a way that every other team shifts in the same direction. Direct swap requires an even number of teams in each race.
  • With no rotation, each team sails the same boat in each race.

The shift from one sail to another happens after each race, with divisions kept separately in the process, but this can be controlled with the Style parameter.

Rotation Style

When creating rotations for more than one division at a time (the standard procedure), it is customary to apply the rules of the rotation, as described above, for the same race number across all the divisions at a time. With the Style you can control at what point does the shift prescribed in the Type occur:

  • With Same for all divisions, the shift happens after all the sails have been assigned for the same race number across all divisions. This style results in all divisions having the same rotation.
  • With Rotate sail at division switch, the shift prescribed in the rotation type happens after every race, regardless of division. In college sailing, this is often referred to as Navy special.
  • With Start sails offset between divisions, known colloquially as the Franny special, the chosen rotation type is applied to the first chosen division. After that, the same rotation type is copied over each other division in turn, with the starting sails shifted by the offset amount.

Races in set

It is often inconvenient to rotate after every race. This parameter allows you specifiy the number of races per set before rotating. Thus, a value of 2 means that each team sails in the same sail for two races at a time before rotating according to the type and style.

(Offset) Amount

This option is only available if using the offset rotation style as discussed above. In that case, the amount to offset can be specified here. Note that the same offset will be applied to each division in turn. Thus, if there are divisions and the offset is 6, then the Division B's rotation will be offset 6 sails from Division A, while Division C's will be offset 12 from A.

Races

It is customary to apply the same rotation to the entire regatta. For convenience, you can specify a subset of races for which to apply the rotation style. This gives you the flexibility to apply a completely unique rotation style as you can tweak the races to be any set of race numbers, separated by commas. Ranges can be specified by using dashes, as shown.

Division order

While most rotations are applied to each division in alphabetical order, some race committees prefer to start rotations with a different division so as to end the regatta with the toughest division sailing last.

To specify a different division order, drag the divisions around until the sequence reflects the desired order. This is particularly useful if using the second or third rotation style, as the first produces the same rotation for each division anyways, as discussed above under Same for all divisions.

Sort sails first

As of version 1.4, sails no longer have to be numbers, they can also include letters. Usually, however, the boats are arranged in order prior to creating the rotation. TechScore will attempt to order the sail values by their natural order, but if no ordering is required, the option is provided as well. This parameter tells TechScore to possibly sort the starting sails in the first race, so that if numbers are used, for example, the shift from one race to another is more logical.

Starting sails

Specify the sails to be used in the lower portion of the pane titled Starting sails. Each team in the regatta is printed with a textbox next to it. You may choose the next available sail by pressing the scroll buttons next to each textbox, or type your own. Note that you will not be allowed to click on Set rotation at the bottom-right of the screen if there are duplicate or missing sail numbers.

Note also that these are the sail numbers in the first race in the first division (as specified by the division order).

Finally, remember to click Set rotation in the lower right corner to actually create the rotation! When you do, the Rotations dialog will open if not already open so that you can admire your work.

Removing a rotation

To remove a rotation, click on the Delete button in the lower left corner. You will be prompted to confirm if finished have been entered. Removing a rotation does NOT affect the scores.
